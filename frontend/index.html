ℹ About TimetablePro

Built with the MEAN Stack — MongoDB, Express, Angular (style), Node.js

🍃

MongoDB

All timetables, subjects, and teacher data are stored in MongoDB — a powerful NoSQL database that handles flexible schemas perfectly.

⚡

Express.js

RESTful API backend built on Express.js with full CRUD routes for timetables, subjects, and teachers. Runs on Node.js.

🔺

Angular (Style)

Component-driven single-page frontend with reactive state, dynamic forms, and real-time timetable generation logic.

🐳

Docker

Fully containerized with Docker Compose. Three services: MongoDB, Express API, and Nginx-served frontend — one command to run all.
